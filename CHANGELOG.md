## 0.1.0 - 2026-06-25

### Added

- Initial audit-packs GitHub Action for mapping IaC security findings to
  compliance controls.
- Framework packs for NIST 800-53, SOC 2, ISO 27001, PCI-DSS, FedRAMP, HIPAA,
  GDPR, and custom organization policies.
- OSCAL assessment-results, control coverage, aggregate SARIF, and inline PR
  review comment outputs.
- Optional AI adjudication pipeline with confidence gating.
